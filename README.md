# Bitloops Tap

## How do I install these formulae?

`brew install bitloops/tap/<formula>`

Or `brew tap bitloops/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "bitloops/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
